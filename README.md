This one of probably several Visual Studio Code Extensions supporting integration with
[codebase-memory-mcp](https://github.com/DeusData/codebase-memory-mcp). This one adds
support for the query language - so users can define custom queries and access them through
a sub-menu. This plugin wires in all the search capabilities provided by codebase-memory-mcp.
